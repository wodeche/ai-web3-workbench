## 今日三条路径

### 🟢 最小路径（1h，必做）

| 时间 | 内容 | 目标 |
|------|------|------|
| 30min | Handbook → [Context](https://aiweb3.school/zh/handbook/ai/context/) | 理解 Context Window / Context Engineering / Memory / Knowledge Base 四个核心概念 |
| 20min | Handbook → [Web3 Network](https://aiweb3.school/zh/handbook/web3/network/) 前半 | 理解区块链是什么、区块、共识、L2 的基本概念 |
| 10min | 记笔记 | 把你理解的 Context 和 Network 用自己话写到下面笔记区 |

**产出**：理解"AI 怎么处理信息"和"Web3 网络怎么运转"

### 🟡 推荐路径（2h）

包含最小路径 + 

| 时间 | 内容 | 目标 |
|------|------|------|
| 20min | Handbook → [RAG](https://aiweb3.school/zh/handbook/ai/rag/) | 理解检索增强生成：为什么 LLM 需要外挂知识库 |
| 15min | Handbook → [Web3 Network](https://aiweb3.school/zh/handbook/web3/network/) 后半 | RPC、链上状态、区块浏览器 |
| 15min | 交叉思考 | 写下：Context 和 Chain-aware Context 有什么关系？RAG 查链上数据和查文档有什么不同？ |

**产出**：把 AI 和 Web3 的知识点连接起来思考

### 🔴 挑战路径（3h+）

包含推荐路径 +

| 时间 | 内容 | 目标 |
|------|------|------|
| 30min | 实践：Context Engineering 练习 | 拿你之前的交易解释器 prompt，重新设计上下文结构——标注每条信息的来源、时效、可信度 |
| 30min | 实践：搭建测试网连接 | 用 Metamask 添加 Sepolia 测试网，去 faucet 领水 |
| 30min | 前瞻阅读 | 快速扫读 Handbook → [Chain-aware Context](https://aiweb3.school/zh/handbook/bridge/chain-aware-context/) 第一节 |

**产出**：升级交易解释器 context 设计 + 测试网准备就绪
